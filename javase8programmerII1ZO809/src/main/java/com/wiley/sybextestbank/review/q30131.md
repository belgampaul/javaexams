Arithmetic Operations with rational numbers
===========================================
**30131**
```
Which lambda can replace the MySecret class? (Choose all that apply) 
 interface Secret { 
   int number(); 
 } 
 
 class MySecret implements Secret { 
   public int number() { 
     return 5; 
   } 
}
```


- 1 caller( -> 5);
- 2 caller(x -> 5);
- 3 caller(() -> 5);
- 4 caller((x) -> 5);
- 5 caller(() -> {5});
- 6 caller(() -> { return 5;});

Answer: *3;6*
