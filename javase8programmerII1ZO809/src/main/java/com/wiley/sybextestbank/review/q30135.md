===========================================
**30135**
```
What is the result of the following code snippet? 
 
 3: for(int i=0; i<5; i++) { 
 4:   System.out.print(1); 
 5:   if(i<2) continue; 
 6:   else break; 
 7: }
```


- 1 1
- 2 11
- 3 111
- 4 1111
- 5 11111
- 6 The code produces no output.
- 7 The code will not compile because of line 5.

Answer: *3*
