Arithmetic Operations with rational numbers
===========================================
**30153**
```
What is the result of the following code snippet? 
 
 3: int x = 9; 
 4: long y = x * (long) (++x); 
 5: System.out.println(y);
```


- 1 –1
- 2 9
- 3 81
- 4 90
- 5 The code will not compile because of line 4.

Answer: *4*
