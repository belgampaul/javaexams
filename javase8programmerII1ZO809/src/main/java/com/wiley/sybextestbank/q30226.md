Arithmetic Operations with rational numbers
===========================================
**30226**
```
What is the result of following code? 
 
 String s = "a"; 
 s += 1; 
 s.concat(s); 
 s.concat("."); 
 System.out.println(s);
```


- 1 a
- 2 a1
- 3 a1a
- 4 a1a.
- 5 An exception is thrown.
- 6 The code does not compile.

Answer: *2*
