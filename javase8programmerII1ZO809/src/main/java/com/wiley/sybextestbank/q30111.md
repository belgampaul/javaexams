Arithmetic Operations with rational numbers
===========================================
**30111**
```
What is the result of the following code? 
 
 public class A { 
   private int i = 6; 
   private int j = i; 
   public A() { 
     i = 5; 
   } 
    public static void main(String[] args) { 
      A a = new A(); 
      System.out.println(a.i + a.j); 
  } }
```


- 1 55
- 2 56
- 3 66
- 4 10
- 5 11
- 6 12
- 7 The code does not compile.

Answer: *5*
