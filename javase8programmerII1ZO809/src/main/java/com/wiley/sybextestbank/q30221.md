Arithmetic Operations with rational numbers
===========================================
**30221**
```
In which order can you assemble the following statements in to make a valid Java program? (Choose all that apply) 
 
A: /* hello world */ 
B: ArrayList list; 
C: import java.util.*; 
D: package abc; 
E: public class Builder { 
F: public void method() { }
```


- 1 A, D, E, B, F
- 2 A, D, C, E, F, B
- 3 D, A, E, B, F
- 4 D, A, E, F, B
- 5 D, C, A, E, B, F
- 6 D, C, A, E, F, B

Answer: *2;5;6*
