===========================================
**30162**
```
What is the result of the following code? 
 
 1: public class DoubleLoopSample { 
 2:   public static void main(String[] args) { 
 3:     char c = 'a'; 
 4:     for(int i=1; i<= 3; i++) 
 5:       for(int j=0; j<= 2; j++) System.out.print(c++); 
 6: } }
```


- 1 abcdefghi
- 2 bcdefghij
- 3 abcdef
- 4 abcabcabc
- 5 The code will not compile.

Answer: *1*
